# 🚕 Ride Booking Data Analysis

Proyek ini bertujuan untuk melakukan **analisis data pemesanan perjalanan (ride booking)** menggunakan pendekatan **Exploratory Data Analysis (EDA)**.
Analisis dilakukan untuk memahami pola penggunaan layanan transportasi online, seperti **metode pembayaran, jarak perjalanan, nilai booking, dan kepuasan pelanggan**.

Hasil analisis diharapkan dapat memberikan **insight yang berguna untuk pengambilan keputusan bisnis** pada layanan ride-hailing seperti Uber, Grab, atau Gojek.

---

# 🎯 Objective

Tujuan dari proyek ini adalah:

* Menganalisis pola pemesanan perjalanan
* Mengidentifikasi metode pembayaran yang paling sering digunakan
* Mengetahui hubungan antara **jarak perjalanan dan nilai booking**
* Menganalisis **rating pelanggan**
* Membandingkan kepuasan pelanggan berdasarkan **jenis kendaraan**

---

# 📂 Dataset

Dataset yang digunakan adalah:

**`ncr_ride_bookings.csv`**

Dataset ini berisi informasi mengenai pemesanan perjalanan pada layanan transportasi online.

Beberapa fitur utama dalam dataset:

* **Booking ID**
* **Ride Distance**
* **Booking Value**
* **Payment Method**
* **Vehicle Type**
* **Customer Rating**

Dataset ini digunakan untuk melakukan **analisis perilaku pengguna layanan transportasi online**.

# 🔄 Data Analysis Workflow

Tahapan analisis dalam proyek ini meliputi:

### 1️⃣ Data Loading

Dataset dimuat menggunakan library **Pandas** untuk dianalisis lebih lanjut.

### 2️⃣ Data Exploration

Melakukan pengecekan data untuk memahami struktur dataset, seperti:

* jumlah data
* tipe data
* distribusi nilai

### 3️⃣ Payment Method Analysis

Analisis dilakukan untuk mengetahui **metode pembayaran yang paling sering digunakan oleh pengguna**.

Visualisasi yang digunakan:

* Pie Chart
* Bar Chart

Hasil ini membantu memahami **preferensi metode pembayaran pelanggan**.

---

### 4️⃣ Ride Distance vs Booking Value

Menggunakan **scatter plot** untuk melihat hubungan antara:

* jarak perjalanan
* nilai booking

Analisis ini membantu memahami apakah **harga perjalanan meningkat seiring bertambahnya jarak perjalanan**.

---

### 5️⃣ Customer Rating Analysis

Distribusi rating pelanggan dianalisis menggunakan **histogram**.

Tujuan analisis ini adalah untuk mengetahui:

* tingkat kepuasan pelanggan
* distribusi rating tinggi dan rendah

---

### 6️⃣ Vehicle Type vs Customer Rating

Menggunakan **boxplot** untuk membandingkan rating pelanggan berdasarkan **jenis kendaraan**.

Analisis ini dapat membantu mengetahui kendaraan mana yang memberikan **pengalaman pelanggan terbaik**.
